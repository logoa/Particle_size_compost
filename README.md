# Particle_size_compost
Amplicon sequencing analysis of composts. Testing the effect of DNA extration methods and particle size on bacterial and fungal communites.

The Analysis consists of the following:

# R project
Compost_particle_size_25.Rproj

# Loading of the environment/packages
Comp_size_25_setup.R

# Preperation of data and calculations
Comp_size_25_calc.R

# Main analysis
Compost_fractions_ITS_0625_alo.R

# Session infos
Session_Info_021225.tyt

# Folders
archive			Contains old R files and outdated summaries of Analysis
sequencing_data		Contains the asv table and tax files output files from the bioinformatical pipeline
output			Contains results of Analysis
data			Contains data
figures			Contains figures
